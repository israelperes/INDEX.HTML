Projeto INDEX.HTML

Este site é um modelo para o Projeto INDEX.HTML, um projeto de desenvolvimento de sites para alunos de programação.

O projeto é dividido em oito encontros, cada um com um tema específico. Os alunos aprenderão sobre HTML, CSS e JavaScript, além de como trabalhar em equipe e gerenciar projetos usando a metodologia Scrum.
