Projeto INDEX.HTML

Sobre o projeto

O Projeto INDEX.HTML é um projeto de desenvolvimento de sites para alunos de programação.

O projeto é dividido em oito encontros, cada um com um tema específico. Os alunos aprenderão sobre HTML, CSS e JavaScript, além de como trabalhar em equipe e gerenciar projetos usando a metodologia Scrum.

Exemplos de sites criados pelos alunos

    Um site sobre animais.

  • Site 2

    Um site sobre receitas.
